# E-Commerce Mobile Application

## Overview
This repository contains the **Flutter frontend** and **Java backend** for an E-Commerce app. Currently, only the **Login Page** and **User Management Microservice** are implemented. Future updates will add product browsing, cart, and order features.

---

## Current Features

### Frontend (Flutter)
- Login Page

### Backend (Java)
- User Management Microservice (registration, login, user data)

---

## Technologies
- Flutter, Dart
- Java, Spring Boot
- MySQL/PostgreSQL (for backend)
